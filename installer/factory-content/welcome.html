Welcome to DrumEngine01 Factory Content

This installer installs DrumEngine01 factory presets only. It does not install the plugin binaries.

What will be installed:

  • Factory Presets: ~/Documents/DrumEngine01/factory/

To install the VST3/AU/AAX plugins, run the separate DrumEngine01 plugin installer.

System Requirements:

  • macOS 10.13 or later
  • Compatible DAW (Digital Audio Workstation)
